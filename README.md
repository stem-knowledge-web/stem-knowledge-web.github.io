# Hi there!

STEM Knowledge is a free, open-source, and community driven knowledge platform that aims to provide no-fluff high quality scientific knowledge to serious and casual learners alike.

## Contributing

See our [contributing](/content/contributing) page here to learn how to contribute via pull-requests.

## Requirement

Please raise issues on our Github repo or email us @ `stem-knowledge-web@gmail.com` to request for topics, learning paths, or to volunteer as content expert for the project.
